Xiaomi Redmi Note 8T

El mejor calidad-precio del mercado

Los puntos principales de este movil son:

  • Pantalla: Pantalla IPS LCD de 6.3 pulgadas con una resolución de 1080 x 2340 píxeles.
  • Procesador: Qualcomm Snapdragon 665 de ocho núcleos.
  • Memoria RAM: 3GB/4GB.
  • Almacenamiento interno: 32GB/64GB, ampliable mediante tarjeta microSD.
  • Cámara principal: Configuración de cámara cuádruple en la parte trasera, que incluye un sensor principal de 48MP, un ultra gran angular de 8MP, un sensor de profundidad de 2MP y un sensor macro de 2MP.
  • Cámara frontal: Sensor de 13MP para selfies.
  • Batería: Batería de 4000 mAh con carga rápida de 18W.
  • Sistema operativo: Android 9.0 (actualizable a versiones más recientes de MIUI).
  • Conectividad: 4G LTE, Wi-Fi 802.11ac, Bluetooth 4.2, USB-C, conector para auriculares de 3.5 mm.
  • Sensor de huellas dactilares: Ubicado en la parte trasera del dispositivo.
